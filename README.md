# LinkedIn Clone Full-Stack Application

This project is a complete LinkedIn clone built using Java Spring Boot (Backend) and React Native/Vite (Frontend) with MySQL as the database.

## Prerequisites
- Java 17
- Maven
- Node.js (v18+)
- MySQL Server

## 1. Database Setup
1. Ensure your MySQL server is running.
2. The Database schema and default tables will be auto-generated by Hibernate (`spring.jpa.hibernate.ddl-auto=update`).
3. If you want to check the schema, see the provided `init_schema.sql`.

## 2. Backend Setup (Spring Boot)
1. Open a terminal and navigate to the `backend` directory: `cd backend`
2. Update the database credentials in `backend/src/main/resources/application.properties` (default is root/root).
3. Run the application using Maven:
   ```bash
   mvn spring-boot:run
   ```
   *The backend will start on http://localhost:8080*

## 3. Frontend Setup (React/Vite)
1. Open a separate terminal and navigate to `frontend`: `cd frontend`
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
   *The frontend will start on http://localhost:5173*
4. Open the link in your browser.

## Postman Sample Requests

You can use the following payloads to test the Backend endpoints via Postman (`http://localhost:8080`):

### 1. Register User
- **URL:** `/api/auth/register`
- **Method:** `POST`
- **Body (JSON):**
  ```json
  {
      "name": "John Doe",
      "email": "john@example.com",
      "password": "password123"
  }
  ```

### 2. Login User
- **URL:** `/api/auth/login`
- **Method:** `POST`
- **Body (JSON):**
  ```json
  {
      "email": "john@example.com",
      "password": "password123"
  }
  ```
*(Note: Use the resulting accessToken as a Bearer Token for subsequent requests)*

### 3. Create Post
- **URL:** `/api/posts`
- **Method:** `POST`
- **Headers:** `Authorization: Bearer <accessToken>`, `Content-Type: text/plain`
- **Body (raw):**
  ```text
  This is my first post on the LinkedIn Clone!
  ```

### 4. Get Feed
- **URL:** `/api/posts`
- **Method:** `GET`
- **Headers:** `Authorization: Bearer <accessToken>`

### 5. Send Connection Request
- **URL:** `/api/connections/request/2`
- **Method:** `POST`
- **Headers:** `Authorization: Bearer <accessToken>`
